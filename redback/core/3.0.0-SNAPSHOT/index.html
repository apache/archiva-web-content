About Apache Archiva Redback

Redback is a authorization and authentication framework used by Archiva. It is developed by the Apache Archiva team.

Project Modules

This project has declared the following modules:

Name Description
Redback :: Authentication API and providers for authenticating users.
Redback :: Authorization API and providers for RBAC based authorization.
Redback :: Configuration Redback Configuration API
Redback :: Commons Common Components used by different Redback modules.
Redback :: Policy Redback security policies
Redback :: Role Based Access Control Redback RBAC API and data model.
Redback :: Core System Redback system components
Redback :: Key Management Management of authentication keys
Redback :: User Management Redback User model
Redback :: Integrations Redback integration modules for security and REST
