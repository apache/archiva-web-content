public abstract void extensionChangePassword (User user)

Extension Point - Change the password of a user.

This method does not check if a user is allowed to change his/her password. Any kind of authorization checks for password change allowed on guest or anonymous users needs to occur before calling this method.

This method does not persist the newly changed user password. That will require a call to updateUser(User).

Parameters
user the user password to validate, remember, and encode.
Throws
PasswordRuleViolationException if the new password violates the password rules
