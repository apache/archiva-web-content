Generated Reports

This document provides an overview of the various reports that are automatically generated by Maven Each report is briefly described below.

Overview

DocumentDescription
changelogGenerated Changelog report from SCM
CheckstyleReport on coding style conventions.
ClirrReport on binary and source API differences between releases

CPD ReportDuplicate code detection.
dev-activityGenerated developer activity report from SCM
file-activityGenerate file activity report from SCM
Maven Surefire ReportReport on the test results of the project.
PMD ReportVerification of coding rules.
Tag ListReport on various tags found in the code.
