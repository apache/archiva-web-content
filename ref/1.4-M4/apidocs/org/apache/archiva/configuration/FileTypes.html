public List<String> getFileTypePatterns(String id)

Get the list of patterns for a specified filetype.

You will always get a list. In this order.

  • The Configured List
  • The Default List
  • A single item list of "**/*"

Parameters:
id - the id to lookup.
Returns:
the list of patterns.
